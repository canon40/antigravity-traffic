## 1단계 진단표 예시

```markdown
| 항목 | 상태 | 근거 |
|------|------|------|
| 직접 경험 | 부족 | 일반 설명만 있고 시공 일화 없음 |
| 일관 주제 | 통과 | 코팅제 비교 주제 유지 |
| 진정성 | 부족 | 협찬 여부 미표기 |
| 구조 | 부족 | 소제목 없음, 장문 단락 |
| 최신성 | 부족 | '작년'만 언급, 구체 월 없음 |
| 이미지 | 해당없음 | 텍스트만 제공 |

**우선 수정**: 진정성 고지 → ## 구조 분리 → 경험 2문장 추가
**권고**: 새 글 재발행 (제목·구조 전면 변경 필요)
```

## 3단계 본문 골격 (복사용)

```markdown
[협찬/내돈내산 고지 — 해당 시 한 줄]

## 도입 — 독자 상황

(공감 2~3문단)

[IMAGE]
(이미지 설명) … 핵심은 아래 표에도 정리.

## 소제목 A — 문제 정의

## 소제목 B — 해결·원리

| 항목 | 내용 |
|------|------|
| … | … |

## 소제목 C — 현장 경험

(직접 경험 2문장 이상, 구체 수치)

## 소제목 D — 정리·팁

(7:2:1 — 홍보 10% 이내)

(독자 질문 1문장?)

태그: …
```

## 4단계 질문 예시

1. 이 글을 쓸 때 실제로 시공·사용한 제품/장소는 무엇이었나요?
2. Before/After에서 눈에 띈 변화를 수치나 기간으로 말씀해 주실 수 있나요?
3. 독자가 가장 많이 물어보는 오해 한 가지는 무엇인가요?
4. 2025~2026 기준으로 바뀐 정보나 가격·규정이 있나요?
5. 협찬·체험·내돈내산 중 어떤 관계였나요?

## 사람이 채워야 하는 20~30%

- 사실 관계(가격, 일정, 제품명) 최종 확인
- 현장 사진 선별·캡션·대체 텍스트
- 말투 미세 조정·오타
- 네이버 에디터에서 발행·공개 범위·예약 설정
- 기존 글 처리(수정 URL 유지 vs 새 글 + 구글 글 안내)
