③ 타사 URL · 벤치마크

네이버 스마트스토어는 자동 수집이 차단(HTTP 429)됩니다.
1) 각 URL 옆 열기 ↗로 페이지를 연다 → 2) 상세 설명 영역 Ctrl+A → 복사 → 3) 아래 본문 붙여넣기 칸에 붙인다 → 4) 타사 URL 분석 다시 클릭
