## 스마트스토어 에디터 붙여넣기 순서

1. **HTML 모드**(또는 소스 보기)에서 `store_detail_bike_퀵.html` 등 전체 복사 → 붙여넣기
2. 이미지 placeholder `<!-- IMG: ... -->` 위치에 실제 이미지 업로드 후 `<img>` 태그로 교체
3. 미리보기에서 모바일 폭 확인
4. 상품명·태그는 쇼핑 검색 탭에 별도 입력 (HTML 본문과 중복 나열 금지)

## img 태그 템플릿 (에디터에서 이미지 삽입 후)

```html
<img src="업로드된URL" alt="듀라코트 퍼마코트 퀵 바이크 발수 코팅" style="width:100%;max-width:860px;display:block;margin:16px auto;" />
```

alt 텍스트에 **바이크·유리막·발수** 중 1개만 자연스럽게 포함 (키워드 스터핑 금지).
