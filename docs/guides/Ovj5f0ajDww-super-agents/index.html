24/7 Super Agent — Base44 영상 로컬 구현

원본: WorldofAI — How to Build 24/7 Claude Agents · login2에서 Ollama로 동일 멀티 에이전트 체인 실행

Base44 Super Agents ↗ LoopReel 스튜디오 ↗ YouTube 학습 허브 ↗

한눈에 보는 에이전트 체인

① Research→② Script→③ Report→④ Gmail→⑤ Schedule

영상처럼 역할별 에이전트를 연결합니다. 한 번에 거대 프롬프트를 쓰지 않고, 리서치 결과가 스크립트·리포트·이메일로 이어집니다.

바로 실행 (3단계)

  1. 터미널: ollama serve (모델 gemma4:e2b 권장)
  2. 프로젝트 루트에서 run_super_agents.bat 실행
  3. data/super_agents/runs/ 의 HTML 열기 → Ctrl+P → PDF 저장
run_super_agents.bat
run_super_agents.bat status
run_super_agents.bat schedule

1Research Agent — 딥 리서치

웹 검색네이버 블로그신뢰도 점수

  • workflow.json 의 research_topics 주제로 검색
  • Ollama가 스토리 3~5개 + [신뢰도 X/10] + 출처 URL 브리핑 생성

2Script Agent — 대본 변환

INTROSEGMENTSCTAOUTRO

  • 브리핑을 YouTube·블로그 촬영용 스크립트로 자동 변환
  • 영상 예시: World of AI Daily Script 형식

3Report Agent — HTML 리포트

영상의 PDF 단계를 스타일 HTML로 대체 (추가 패키지 없음)

  • 브리핑 + 스크립트 + 출처 링크가 한 페이지에 정리
  • 브라우저에서 인쇄 → PDF로 저장

4Notify Agent — Gmail (선택)

.env 설정 후:

SUPER_AGENT_GMAIL_USER=you@gmail.com
SUPER_AGENT_GMAIL_APP_PASSWORD=앱비밀번호
SUPER_AGENT_EMAIL_TO=받을주소@gmail.com
run_super_agents.bat run --email
Gmail 2단계 인증 + 앱 비밀번호 필요
