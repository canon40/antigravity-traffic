Google FLOW로 뮤직비디오 10분 만들기

원본: GPT PARK — AI로 뮤직비디오 10분 컷 · 프로젝트에 자막 학습 완료 (Gen 10)

Google FLOW ↗ CapCut ↗ LoopReel 쇼츠 스튜디오 ↗ YouTube 학습 허브 ↗

한눈에 보는 전체 흐름

① 캐릭터→② 배경→③ FLOW 클립→④ 프레임 영상→⑤ Suno 음악→⑥ MV 일괄 생성→⑦ 루프(선택)→⑧ CapCut 합성

텍스트 몇 줄 → AI 음악 → 캐릭터가 가사에 맞게 움직이는 MV. 제작비 0원(무료 할당량·도구 기준).

1캐릭터 에셋 (Google FLOW)

비율 4:3생성 4장Nano Banana Pro

고전적인 아름다움을 간직한 20대 한국 여성의 얼굴
  • 각 이미지 이름 수정 · 프로젝트명 캐릭터_여성 등으로 저장
  • 입력창 @ → 에셋 불러오기로 라이브러리에서 재사용

2배경 에셋

비율 16:9생성 4장

평화롭고 낭만적이면서 아름다운 들판
  • 프로젝트명 배경으로 저장

3에셋으로 영상 클립 만들기

프롬프트 우측: 동영상 탭 + 에셋 · 16:9 · 생성 1개 · 길이 6초

  1. @ 로 저장한 캐릭터 이미지 1장
  2. @ 로 배경 1장
  3. 프롬프트: 캐릭터가 배경을 걸어가는 장면 → 실행
영상 생성은 크레딧이 듭니다. 테스트는 1개·6초부터.

4프레임(시작·끝) 영상 — 자연스러운 동작

우측 옵션: 동영상 프레임 · 시작·종료 이미지 드래그 등록

OmniFlash 모델은 프레임 미지원 → Veo 3.1 Lite로 변경 후 다시 등록.

짧은 프롬프트만 입력하면 시작·끝 정지컷 사이로 미소·손 흔들기 등이 보간됩니다.

5FLOW 안에서 편집·확장

  • 타임라인: 클립 앞·뒤 끝을 드래그해 트림
  • 영상 끝 + → 확장 → 예: 나비를 만나서 즐겁게 논다

6음악 만들기 (Suno)

영상 설명란 GPT 공식 사이트의 「뮤직비디오 만들기」도구 사용:

  1. AI 가사 생성 — 키워드·장르 입력 → 제목·스타일·가사 생성
  2. 제목·스타일·가사를 Suno에 붙여넣기 → 음악 생성
  3. 완성 MP3 다운로드

7MV 스토리보드 → 일괄 영상

  1. 비주얼·시나리오: 장면 수 (예: 2)
  2. FLOW 이미지 불러오기 → 캐릭터 에셋 선택 · 화풍 Cinematic 등
  3. 스토리보드 생성 → 가사 분석 후 프레임 자동 생성
  4. 일괄 처리 → 전체 시네마틱 영상 생성
  5. ZIP 다운로드
FLOW만으로는 완벽한 seamless loop가 어렵습니다. 다음 단계 참고.

8심리스 루프 (선택)

GPT 공식 사이트 → 프로그램 → Seamless Loop Studio

  1. 영상 업로드
  2. 출력 설정: 루프 방식 타입 2 (가장 자연스러운 매칭)
  3. 구간 분석 시작

9CapCut 최종 합성

  1. 루프 클립 + Suno 음악 가져오기
  2. 오디오 트랙에 음악 먼저 배치
  3. 비디오 트랙에 클립 순서대로 배치
  4. 음악 길이 넘는 영상 구간 자르기
  5. 전환 → 오버랩 효과 (클립 경계)
  6. 처음부터 재생 확인 → 내보내기

LoopReel(이 프로젝트)과 함께 쓰기

  • 쇼핑·제품 쇼츠: 쇼츠 스튜디오 ①콘티 → ②보드 → FLOW 프롬프트 복사 → ④미리보기 MP4
  • 실사 B-roll: 스토리보드 각 장면 Google FLOW ↗ 버튼
  • 최종 퀄리티: 이 가이드처럼 FLOW 클립 + CapCut (또는 MV 도구 일괄 생성)
